How This Site Ships

This portfolio is itself a DevOps project — version-controlled and deployed via CI/CD to GitHub Pages.

.github/workflows/deploy.yml
name: 🚀 Deploy Portfolio

on:
  push:
    branches: [main]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/configure-pages@v5
      - uses: actions/upload-pages-artifact@v3
        with: { path: "." }
      - uses: actions/deploy-pages@v4
      - name: 🏥 Health Check
        run: curl -f https://priteshksr.github.io/pritesh-portfolio/
